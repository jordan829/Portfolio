Tacode Mobile Banking

  • View on Github

This is an application that was developed for a software engineering course at UCSD over the course of 10 weeks. I worked in a group of 11 and was made personally responsible for implementing the user interface. I was given mockups of interface designs and was tasked with turning them into a reality. The UI was implemented using Java for pages that required dynamic creation, such as each individual customer's accounts page, as well as XML for basic page structure. Our group worked on an iterative, Agile development cycle and had regular scrum meetings where we discussed the state of the application and our own personal progress.
